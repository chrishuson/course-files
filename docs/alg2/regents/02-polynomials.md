# Algebra II Regents Problems: Polynomials

Which of the following polynomials has a degree of 3?

(1) $2x^2 + 3x - 1$
(2) $4x^3 - 2x^2 + 5x + 1$
(3) $x^4 - 3x^2 + 2x$
(4) $5x^2 - 4x + 3$
